其他按钮样式

圆角按钮

class 中增加 .btn-round。

带图标的按钮

class 中增加 .btn-label，button 元素中增加 label 元素。

指定按钮宽度

可使用 .btn-w-xs, .btn-w-sm, .btn-w-md, .btn-w-lg 和 .btn-w-xl 类指定按钮宽度。

按钮大小

使用 .btn-lg, .btn-sm 和 .btn-xs 类指定按钮的大小。

块级元素按钮

加上 .btn-block 类可以使得按钮变成一个块级元素按钮。
